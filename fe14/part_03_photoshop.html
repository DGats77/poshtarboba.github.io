Програма Adobe Photoshop, порізка макета.

Налаштування Adobe Photoshop.

Запустіть програму Adobe Photoshop, скачайте і відкрийте макет сайту з продажу футболок (jpg).

Ctrl + H - прибрати блакитні лінії (направляючі).

Скріншот головного вікна Adobe Photoshop.

Для порізки спочатку нам необхідно трохи налаштувати фотошоп.

Першим чином включіть панель інструментів роботи з текстом:

Window > Character

Ми будемо використовувати панелі Layers, Character, Paragraph, решту панелей можна прибрати, щоб вони нас не відволікали і не займали простору екрану.

Виберіть інструмент Move Tool (зліва перша іконка зверху), поставте галочку "Auto-Select" і в списку виберіть Layer. Це дозволить кліком мишки активізувати той шар, по якому елементу ми клацнули.

Налаштування інструменту Move Tool.

Налаштуйте використання одиниць вимірювання, встановивши значення Pixels в перших двох пунктах:

Edit > Preferences > Units & Rules...

Налаштування одиниць вимірювання.

За потреби можна переналаштувати розміщення тимчасових файлів фотошопа:

Edit > Preferences > Performance...

Налаштування розміщення тимчасових файлів.

Основні інструменти верстальника.

Інструменти верстальника.

Move Tool, стрілочка v - вибір елементів, шарів, їх переміщення по канві.

Slice Tool, скальпель c - вирізання елементів з макету (створює рамочки, які в подальшому будуть вирізані).

Eyedropper Tool, піпетка i - визначення кольору (не використовувати для тексту!).

Horizontal Type Tool, текст t - визначення параметрів шрифта, кольору тексту.

Hand Tool, рука h, Space - переміщення канви, якщо вона не влізає на екран.

Zoom Tool, зум z, ctrl + +/-, alt + scroll, - збільшення/зменшення маштабу канви.

Панель шарів.

Opacity, Fill - прозорість шару.

Око - видимість шару, включити/відключити шар.

Порізка макета.

Запустіть програму Adobe Photoshop і відкрийте макет сайту з продажу футболок (jpg).

Для вирізання елементів використовується інструмент "Скальпель" (гаряча клавіша c).

При вирізанні слайсів одразу задавайте їм імена (подвійний клік по слайсу чи клік із зажатим ctrl).

Якщо в макеті є подібні елементи - виріжте один, дайте йому ім'я, далі - копіюйте цей слайс, тримаючи клавішу alt.

В першу чергу вирізайте фотографії, іконки, логотипи, відключіть всі фони, щоб тло було "прозоре" (шахматний фон). Збережіть ці зображення, включіть всі фони і вимкніть шари з текстом та вирізаними зображеннями, розмітьте і збережіть всі фони.

Щоб видалити всі слайси - можна скористатися пунктом меню View > Remove slices або Просмотр > Удалить фрагменты.

Збереження зображень, типи файлів.

Для збереження вирізаних слайсів скористайтеся пунктом меню чи гарячими клавішами:

File > Save For Web...
Файл > Сохранить для Web и устройств...
Ctrl + Alt + Shift + S

У цьому вікні потрібно підібрати формати файлів, якість стиснення для кожного слайса.

Для зручності - виділяйте декілька подібних слайсів, утримуючи shift.

Формати файлів:

JPG - для фотографій, великих фонових зображень;

PNG - для елементів дизайну, іконок, всіх елементів з прозорістю;

GIF - для дуже маленьких елементів з малою кількістю кольорів.

Для формату JPG вибирайте якість 60-70 для невеликих, неякісних зображень, розпливчастих, блідих фонів, 80-90 - для чітких, крупних, хороших зображень.

Червоні кольори більше за інших створюють "артефакти" в форматі стиснення JPEG, тому при значних кількостях червоного кольору та різких переходів до контрасних до нього кольорів бажано підвищити якість збереження.

Артефакти JPEG.

Імена файлам потрібно давати з розумом, таким чином, щоб подібні файли знаходилися поруч в списку файлів. Для цього використовуються префікси.

Іменування файлів, порівняйте ліву та праву панелі.
